== Description ==
Career Quiz Plugin allows users to take a quiz and receive career recommendations based on their answers. It provides an easy-to-use shortcode for embedding the quiz into any post or page.

== Installation ==
1. Upload the `career-quiz` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure quiz questions in the admin panel under "Career Quiz Settings."
4. Use the `[career_quiz]` shortcode to display the quiz on any page or post.

== Frequently Asked Questions ==
= How do I add new questions? =
Go to the Career Quiz settings page in the WordPress admin panel and enter questions in JSON format.

= Can I style the quiz? =
Yes, you can modify the `quiz-style.css` file included in the plugin.
